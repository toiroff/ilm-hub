# ILM Hub

Premium information, storytelling, and registration website for ILM Hub.

## Stack

- React + TypeScript (Vite)
- Tailwind CSS v4
- Framer Motion
- React Router

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Pages

- `/` — Home (hero, stats, about, impact, challenges, seasons, timeline, gallery, testimonials, contact)
- `/projects` — Project showcase with search & filters
- `/events` — Upcoming events
- `/seasons/:id` — Individual season story pages
- `/register` — Season 5 registration
